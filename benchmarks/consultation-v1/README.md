# CareVR Gold Standard Benchmark v1

## Purpose

This consultation benchmark defines the expected behaviour of the
CareVR Clinical Consultation Understanding Engine.

Every enhancement to the extraction engine should be validated
against this benchmark before release.

The objective is not OCR accuracy.

The objective is accurate clinical understanding.

---

## Clinical Sections

The benchmark validates the following sections.

1. Consultation Vitals

2. Presenting Complaints

3. Past Medical History

4. Examination Findings

5. Doctor Assessment

6. Medicines Prescribed

7. Investigations Ordered

8. Doctor Instructions

---

## Success Criteria

CareVR should correctly identify, classify and display each
clinical section so that a caregiver can quickly review,
confirm and save the consultation.

The benchmark should improve over time.

Regression is not acceptable.
